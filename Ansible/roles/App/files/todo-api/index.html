Todo API

Static test console

Open this page from your server and click the links below to verify the API.

Base URL

http://<server-ip>:3000

Quick check

  1. Replace <server-ip> in the links below.
  2. Open /todos and confirm JSON is returned.
  3. Use the curl samples to create and update todos.

Endpoints

  • GET /todos
  • POST /todos
  • GET /todos/:id
  • PUT /todos/:id
  • DELETE /todos/:id

Open in browser

Update the server IP before clicking.

Open /todos Open /todos/:id

Payload shape

{
  "title": "Ship new API",
  "description": "Add UI test page",
  "priority": "medium",
  "completed": false
}

Sample requests

Create

curl -X POST http://<server-ip>:3000/todos \
  -H "Content-Type: application/json" \
  -d '{"title":"First todo","description":"Test from curl","priority":"high"}'

List

curl http://<server-ip>:3000/todos

Update

curl -X PUT http://<server-ip>:3000/todos/<id> \
  -H "Content-Type: application/json" \
  -d '{"completed":true}'

Delete

curl -X DELETE http://<server-ip>:3000/todos/<id>

Expected response

{
  "status": "success",
  "data": {
    "todos": [
      {
        "_id": "64f...",
        "title": "First todo",
        "description": "Test from curl",
        "priority": "high",
        "completed": false
      }
    ]
  }
}
