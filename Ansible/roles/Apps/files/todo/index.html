Test console for CRUD operations

How to Test CRUD

Use curl from your terminal to test API endpoints

  • List all curl http://localhost:3000/todos
  • Create curl -X POST http://localhost:3000/todos -H "Content-Type: application/json" -d '{"title":"My task","description":"Details","priority":"high"}'
  • Update curl -X PUT http://localhost:3000/todos/ID -H "Content-Type: application/json" -d '{"completed":true}'
  • Delete curl -X DELETE http://localhost:3000/todos/ID
All requests are made from the browser. No server-side changes required.
